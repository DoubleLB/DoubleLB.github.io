# 本地路由

这个站点是 Vue 单页应用，二级、三级页面都在同一个 `index.html` 和 `assets/index-C-sc3Fg2.js` 里，通过 `#/路径` 切换。

本地服务器启动后可以直接访问：

- `http://127.0.0.1:8081/#/`
- `http://127.0.0.1:8081/#/new`
- `http://127.0.0.1:8081/#/me`
- `http://127.0.0.1:8081/#/xjcx`
- `http://127.0.0.1:8081/#/edit`
- `http://127.0.0.1:8081/#/account`
- `http://127.0.0.1:8081/#/about`
- `http://127.0.0.1:8081/#/logout`
- `http://127.0.0.1:8081/#/onlineexam`

说明：

- 以上路由来自下载后的 JS bundle 末尾路由表。
- 不属于 `https://xuexin.pages.dev/#/` 这个单页应用内部的外部跳转页面没有下载。
- `index.html` 已改为本地相对资源路径，并移除了 Cloudflare 统计脚本，方便本地运行和修改。
